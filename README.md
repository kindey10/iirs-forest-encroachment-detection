# Forest Encroachment Detection in Udham Singh Nagar Using Temporal Satellite Imagery and Hybrid ViT-LSTM Model

This project is part of my IIRS internship work. The aim is to study possible forest encroachment in Udham Singh Nagar, Uttarakhand using temporal satellite imagery from Google Earth Engine and a hybrid Vision Transformer (ViT) - Long Short-Term Memory (LSTM) model.

## Current Plan

1. Collect Sentinel-2 satellite imagery using Google Earth Engine.
2. Prepare yearly or seasonal temporal images for Udham Singh Nagar.
3. Create a baseline change detection method using NDVI.
4. Prepare satellite image patches for model training.
5. Build and evaluate a hybrid ViT-LSTM model.
6. Compare the proposed model with simpler baseline methods.
7. Generate maps, graphs, and a final research report.

## Tools

- Google Earth Engine
- Sentinel-2 satellite imagery
- Python
- Google Colab
- Vision Transformer
- LSTM
